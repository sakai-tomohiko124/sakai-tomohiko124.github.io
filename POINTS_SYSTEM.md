# ポイント（金額）システム概要

## 基本設定
- **初期ポイント**: 20,000円
- **システム**: LocalStorageで管理、ブラウザ間で共有

## ポイントが減る行動（コスト）

| 行動 | 金額 | 説明 |
|------|------|------|
| クイズ回答 | -100円 | index2.htmlでクイズに回答するたび |
| チャット送信 | -100円 | index3.htmlでチャットメッセージを送信するたび |
| 用語検索 | -50円 | home.htmlで用語を検索するたび（5秒間隔制限あり） |

## ポイントが増える行動（報酬）

| 行動 | 金額 | 説明 |
|------|------|------|
| eラーニング参加 | +2,000円 | TMlearning.htmlにアクセス（1日1回） |
| アンケート回答 | +1,000円 | アンケート.htmlでアンケートに回答 |
| 物語閲覧 | +1,000円 | story.htmlで物語を読む（各物語1回のみ） |
| 動画視聴完了 | +1,000円 | movie.htmlで動画を30秒以上視聴（各動画1回のみ） |
| 天気情報確認 | +500円 | weather.htmlで天気情報を確認 |
| 電車情報確認 | +500円 | train.htmlで電車情報を確認 |
| マニュアル閲覧 | +300円 | index5.htmlでマニュアルを閲覧 |

## 実装ファイル

### JavaScript
- `static/points.js` - ポイント管理のコアシステム
- `static/points-header.js` - 共通ヘッダーコンポーネント（ポイント表示タブ）

### CSS
- `static/responsive-header.css` - ポイント表示タブとモーダルのスタイル

## 各HTMLファイルでの実装

### 統合済み
- ✅ TMlearning.html - eラーニングアクセスポイント付与
- ✅ movie.html - 動画視聴ポイント付与
- ✅ home.html - 用語検索ポイント減算

### 統合予定
- ⏳ index2.html - クイズ回答ポイント減算
- ⏳ index3.html - チャット送信ポイント減算
- ⏳ story.html - 物語閲覧ポイント付与
- ⏳ weather.html - 天気確認ポイント付与
- ⏳ train.html - 電車情報確認ポイント付与
- ⏳ index5.html - マニュアル閲覧ポイント付与
- ⏳ アンケート.html - アンケート回答ポイント付与

## ポイント表示
- 画面右上に固定表示
- クリックで詳細モーダルを表示
- ポイント変動時にアニメーション通知

## 履歴機能
- 最新100件のポイント変動履歴を記録
- モーダルから履歴を閲覧可能

## レスポンシブ対応
- PC: 画面右上にポイントタブ表示
- スマホ/タブレット: コンパクト表示に自動調整
